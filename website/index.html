FAQ

A few quick answers.

What is HYDRV?

HYDRV is the new name for the project that used to be xManager. It is still the same Android app for browsing releases and downloading files through signed links.

Is this the full public website?

Yes. This is the current public landing page, and future updates can expand it into a release archive, build details, and more project context.

Can this be deployed to hydrv.app?

Yes. It is a static site and can be deployed to Cloudflare Pages, Netlify, GitHub Pages, or any other static host.
