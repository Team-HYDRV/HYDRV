FAQ

What is HYDRV?

HYDRV is a release browser for Android apps, built to keep downloads, versions, and installs easy to follow.

Where do downloads come from?

The app uses its built-in catalog by default, and advanced users can swap to a custom backend source when needed.

Is it open source?

Yes. Releases, issue tracking, and contributor history live on GitHub.
